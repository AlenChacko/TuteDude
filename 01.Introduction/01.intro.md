**What is Python?**
    Python is a high-level,general purpose interpretted programming language.

**Features**
    _High-level_ -> Easy to read and close to human language
    _General Purpsoe_ -> Can be used for all purpose, AI,Machine Learning,Game Developing, Web Development,Automation.
    _Interpretted_ -> Executed during runtime line by line by an interpretter
    _Dynamically Typed_ -> Python determines the type of variables automatically during runtime.
    _Cross-Platform_ -> Can write code in Windows,Linux,Mac os
    _Multiple Paradigm_ -> Support multiple coding styles like object oriented,procedural,functionals
    _Large Eco-system_ -> Python has vast third party eco-system like frameworks and libraries.

**History**
    Python was created by _Guido van Rossum_ a dutch programmer.The development started in 1989, by a _Hobby_ project. But Python's first public release was in _1991_. During the time Python has many versions, the latest version of python is Python-3

**Why Python is famous?**
    It has clean syntax,huge libraries,beginner friendly and powerfull

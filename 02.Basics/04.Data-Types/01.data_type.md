**What are data types?**
    Every values used in python has a type of data.
    Python is dynamically typed, so that the data type is automatically detected.
    The type of data is decided at the run time.

**Categories of Data Type**
    *1.Primitive data types*
    These will store a single value
    int, float, str, complex, bool, NoneType
    *2.Non-Primitive data types*
    These will store multiple values
    list, tuple, set, dict, range, bytes, bytearray, frozenset

**Dynamic Typing**
    Can change the data of the variable any time

**Type Conversion**
    int()
    float()
    str()
    list()
    tuple()
    set()

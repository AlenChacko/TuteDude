**What are keywords?**
    Keywords are reserved words in Python that have special meaning and cannot be used as variable names.

**List of Python Keywords**
    | Keyword    | Meaning                     |
    | ---------- | --------------------------- |
    | `if`       | Conditional execution       |
    | `elif`     | Else-if condition           |
    | `else`     | Final alternative condition |
    | `for`      | Loop over sequences         |
    | `while`    | Loop until condition false  |
    | `break`    | Exit loop                   |
    | `continue` | Skip to next iteration      |
    | `pass`     | Do nothing (placeholder)    |
    | `True`  | Boolean true  |
    | `False` | Boolean false |
    | `None`  | No value      |
    | `and`   | Logical AND |
    | `or`    | Logical OR  |
    | `not`   | Logical NOT |
    | `def`    | Define function            |
    | `return` | Return value from function |
    | `lambda` | Anonymous function         |
    | `class`  | Define class               |
    | `with`   | Context manager            |
    | `try`     | Error testing block |
    | `except`  | Catch error         |
    | `finally` | Always execute      |
    | `raise`   | Throw an exception  |
    | `assert`  | Debug check         |
    | `import` | Import module          |
    | `from`   | Import specific part   |
    | `as`     | Rename module/variable |
    | `global`   | Use global variable inside function        |
    | `nonlocal` | Use variable from outer (non-global) scope |
    | `in`    | Membership / iteration |
    | `is`    | Identity comparison    |
    | `yield` | Return from generator  |
    | `del`   | Delete object          |

**Why Keywords Are Important?**
    Because they:
    Define the structure of your program
    Control logic
    Handle errors
    Create functions
    Control scopes
    Manage loops
    Import modules

**Difference between keywords and identifiers**
    | Keywords           | Identifiers            |
    | ------------------ | ---------------------- |
    | Predefined words   | User-defined names     |
    | Cannot be reused   | Can be created by user |
    | Have fixed meaning | Meaning chosen by user |
    | 35 total           | Unlimited              |

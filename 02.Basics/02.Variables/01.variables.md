**What is a variable?**
    It is a name that stores a value in memory

**How Variables Work Internally (Very Important)**
    Python do not store values directly, instead they store references to objects in memory
    a=100
    Python creates an integer object in memory, a simply points to that object

**Creating a Variable**
name=value
num=10
text="Well Done"

**Rules for Naming Variables (Very Important)**
    ✔ Allowed:
    Letters → a-z, A-Z
    Numbers → 0-9
    Underscore → _
    ❌ Not allowed:
    Cannot start with a number
    → 2value ❌
    No spaces
    → my variable ❌
    No special characters
    → my-value, @name, age# ❌

**Assigning Multiple Variables**
a, b, c = 10, 20, 30
x = y = z = 50

**Deleting a variable**
x=100
del x


**Dynamic Typing**
x = 10
x = "Hello"

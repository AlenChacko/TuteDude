**What is exceptions?**  
Exception is the error that occurs when the programs running
These errors will crash the program and the remaining code will not run. It will show the error message.Exception handling means controlling this crash, if any error occures, we prevent the crash by saying do something else.

**Types of Errors**  
1. Compile time errors:
    These are occures during the compile time, these errors are the syntax errors, indentation errors. These errors are not to handle, these should be fixed.
2. Exceptions:  
     Errors during execution, it should be handled to prevent crash

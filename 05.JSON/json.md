**What is JSON**  
It stands for JavaScript Object Notation, it is a text based data format used to store and exchange data  

**Why Python needs JSON**  
Python cannot directly read or write JSON, it needs a converter  
| Python    | JSON   |  
| --------- | ------ |  
| dict      | object |  
| list      | array  |  
| tuple     | array  |  
| str       | string |  
| int/float | number |  
| True      | true   |  
| False     | false  |  
| None      | null   |
